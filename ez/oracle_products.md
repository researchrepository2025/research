# Oracle AI Platform - Product Announcements
## Date Range: August 1, 2025 - January 13, 2026

### Executive Summary
Oracle announced major AI building platform capabilities from August 2025 through January 2026, centered primarily around Oracle AI World 2025 (October 13-16, Las Vegas) - the rebranded Oracle CloudWorld event. Key announcements included Oracle AI Database 26ai with integrated AI capabilities, OCI Zettascale10 supercomputer infrastructure, Oracle AI Data Platform for enterprise AI deployment, Oracle Fusion Applications AI Agent Marketplace with 100+ agents, and expanded partnerships with NVIDIA (Blackwell GPUs), OpenAI (Stargate project), Meta (Llama 4), Google (Gemini 2.5), and xAI (Grok models). Oracle positioned itself as a comprehensive AI building platform provider spanning infrastructure, data management, model access, and application integration.

### October 14, 2025 - Oracle AI Database 26ai
**Event:** Oracle AI World 2025
**Category:** Oracle Database AI

**Description:**
Oracle released Oracle AI Database 26ai, a major update to its flagship database that integrates AI capabilities throughout its core architecture. This is a long-term support release replacing Oracle Database 23ai. Advanced AI features such as AI Vector Search are included at no additional charge. Key features include Oracle Autonomous AI Lakehouse supporting Apache Iceberg format, Unified Hybrid Vector Search combining AI with relational and text searches, MCP Server Support for AI agents, built-in data privacy protection with row, column, and cell-level controls, quantum-resistant encryption (ML-KEM) for data-in-flight protection, and in-database AI agent builder with no-code visual platform. The update requires no database upgrade or application re-certification for existing 23ai users. Throughout 2025, Oracle released regular quarterly updates for Database 23ai including Release Update 23.7 in January 2025, Release Update 23.8 in April 2025, and Release Update 23.9 in July 2025, each featuring updates to Oracle AI Vector Search capabilities.

**Source Quote:**
> "By architecting AI and data together, Oracle AI Database makes 'AI for Data' simple to learn and simple to use."

**Source:** [Oracle AI Database 26ai Powers the AI for Data Revolution](https://www.oracle.com/news/announcement/ai-world-database-26ai-powers-the-ai-for-data-revolution-2025-10-14/) | Published: October 14, 2025

### October 14, 2025 - OCI GPU Scanner (General Availability)
**Event:** Oracle AI World 2025
**Category:** OCI AI Infrastructure

**Description:**
Oracle Cloud Infrastructure (OCI) GPU Scanner is now generally available. OCI GPU Scanner is a dedicated solution that provides observability, health checks, and performance monitoring for GPU workloads. OCI GPU Scanner brings a vendor-neutral approach - supporting current and future GPU models without firewall exceptions. It integrates seamlessly with cloud-native open-source tools, providing a unified command center for customizable dashboards and automated workflows, allowing users to track everything from temperature and processing power to model flops utilization (MFU). OCI GPU Scanner minimizes downtime and false positives through comprehensive health checks, baseline comparisons, and automated diagnostics. This solution provides comprehensive, active performance and passive health checks for both NVIDIA and AMD GPUs on OCI.
