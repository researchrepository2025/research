## RESEARCH NOTE

Exhaustive search across multiple queries ("Canadian AI coding assistant companies 2025", "AI developer tools startups Canada", "Canadian AI code generation companies", BetaKit archives, Seedtable Canada directory, Canada.ai directory) found **no dedicated Canadian-headquartered AI coding assistant or code generation product companies** comparable to Cursor, Windsurf, or GitHub Copilot.

All major AI coding assistant brands (Tabnine — Tel Aviv, Israel; Qodo/CodiumAI — Tel Aviv, Israel; Pieces for Developers — Cincinnati, OH, USA; Mutable.ai — San Francisco/Dover, USA; Parity — San Francisco, USA) confirmed as non-Canadian headquartered.

The closest Canadian company in this segment is **Cohere**, which provides LLM APIs with code-assistance capabilities as part of a broader enterprise AI platform — not a pure-play coding assistant. It is included below as the primary Canadian entrant.
